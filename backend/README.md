# Digital Media Social API

A social-media-style backend API built with Django REST Framework and JWT authentication.

## Features
- User registration, login, logout (JWT)
- User-owned media entries
- Public media feed
- Genre tagging (Many-to-Many)
- Reviews system
- Secure CRUD with ownership checks

## Tech Stack
- Django
- Django REST Framework
- Simple JWT
- SQLite (development)
- Postman for API testing

## Setup
```bash
cd backend
python manage.py runserver
